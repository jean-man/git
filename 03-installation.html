Installation

Dans votre terminal, tapez la commande “git” puis suivez le programme d’installation.

Vous pouvez ensuite configurer git à l’aide des commandes suivantes :

git config --global user.name "John Doe"
git config --global user.email johndoe@example.com

Installation

Pour créer un nouveau repository (dépôt), positionnez-vous dans un dossier et utilisez la commande suivante :

git init

Pour récupérer la copie d’un repository git sur lequel vous souhaitez contribuer, utilisez la commande suivante :

git clone git://github.com/schacon/simplegit-progit.git

Installation

Si l’opération précédente c’est bien déroulée, vous devriez voir un nouveau dossier “.git” à l’emplacement où vous avez effectué la commande.

La commande suivante devrait vous afficher l’état de votre dépôt git :

git status
