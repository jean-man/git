Présentation

Présentation

  • Système de gestion de versions
  • Décentralisé
  • Optimisé pour l’utilisation des branches
  • Créé par Linus Torvald
  • Plateforme en ligne (Github)

Avantages

  • Simplifier le partage de code source
  • Gérer des versions
  • Bien séparer les développements
  • Sauvegarder son code source
  • Annuler les dernières modifications
  • Rendre son code Open Source (Github)
  • Valider la conformité du code source (Hook)
  • Uploader ses fichiers sur un serveur
  • Trouver la personne responsable d’un bug
