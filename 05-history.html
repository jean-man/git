Historique

La commande git log permet de consulter l’historique d’un dépôt git.

Executez les commandes suivantes sur le dépôt simplegit :

git log
git log -p -2
git log --stat
git log --pretty=oneline
git log --graph
git log --since=2.weeks
Chaque commit possède un identifiant unique (une suite de 40 caractères alphanumériques)
