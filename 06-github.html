Github

Allez sur le site internet github.com et créez vous un compte.

Créez ensuite un nouveau dépôt puis récupérer son contenu sur votre poste en local à l’aide de la commande git clone.

Vous pouvez ensuite utiliser la commande suivante pour afficher la liste des dépôts distants :

git remote
Utilisez git remote -v pour afficher également les urls des dépôts

Github

Positionnez-vous dans le nouveau dossier créé puis exécutez les commandes git add, git commit puis git push pour envoyer sur le dépôt distant vos modifications :

git push origin master

Ici, “origin” correspond au nom du dépôt distant et “master” à la branche que l’on souhaite pusher.

Vous pouvez consulter l’état d’un dépôt distant à l’aide de la commande git remote show origin

Github

Il est possible de travailler avec plusieurs dépôts distants en les ajoutant (git remote add), en les renommant (git remote rename) ou en les supprimant (git remote rm).

git remote add simplegit git://github.com/schacon/simplegit-progit.git
git fetch simplegit
git remote rename simplegit sgit
git remote rm sgit
