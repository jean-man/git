Tags

Git permet de créer des tags afin le plus souvent, de pouvoir préciser la version d’un projet dans l’historique.

Utilisez la commande suivante pour afficher l’ensemble des tags d’un dépôt :

git tag

Tags

Vous pouvez ensuite créer un nouveau tag avec la commande suivante :

git tag -a v2.1 -m 'Version 2.1'

Vous pouvez également ajouter un tag sur un commit précédent :

git tag -a v2.1 -m 'Version 2.1' 150ce56

Et ensuite envoyer sur le dépôt distant :

git push origin --tags
